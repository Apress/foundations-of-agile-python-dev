Test Table Parameters

The April 8, 2004 Fitnesse release added the ability to put parameters on the first line of a table. Unfortunately, all of the ATs for this and other features are on Fitnesse, so they aren't availible in batch.

This just compares the second line, cell by cell, against the parms array that we get from Fixture. It's probably the simplest fixture imaginable.

eg.ParmTest This Is A Test
eg.ParmTest This Is A Test

If this worked, the first cell should be white, and the other four green.
