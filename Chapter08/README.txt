The directory "buildmaster" contains the buildbot master config
for Chapter07. The only changes is the addition of a step to run
coverage tests.  The directory is rooted at '/' on build master.

The directory "rsreader" contains a new version of the project supporting the coverage calls from the new buildbot configuration.

The directory "svnhost" contains the pre-commit hook created in the section on using subversion to check your code.  The directory is rooted at '/' on the subversion server.
