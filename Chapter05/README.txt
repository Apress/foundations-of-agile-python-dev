The directory 'buildmaster' contains the configuration files for the
linux buildbot master.  The directory is root at '/' on the build master.

The directory 'buildslave' contains the configuration files for the
corresponding buildbot slave, and it is also rooted at '/' on the build slave.

The directory 'rsreader' contains the tiny harnessed project developed
in Chapter 4.

The director 'svnhost' contains the post-commit hook connecting buildbot
and subversion.  It is rooted at '/' on the subversion server.
